1. **generateSelfReplicatorProgram()の完全実装**

   - 成長フェーズ: HULL容量を100から200まで拡張

   - エネルギー回収と次サイクルへの移行

2. **正しい命令フォーマットの使用**
   - UNIT_MEM_READ: 結果はBレジスタに格納、MOV_BAでAに移動

   - バイト配列: [opcode, 0x00, unitId, memAddr]

3. **テンプレートマッチング用パターン**
   - 10101010: 成長フェーズの待機ループ
   - 01010101: 娘HULL生産の待機ループ

4. **ジャンプオフセット自動計算**

   - 後方参照によるオフセット修正

5. **包括的なテストスイート**
